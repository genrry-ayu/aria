平台战略
SOURCE
AI & I PODCAST
模型与「马具」深度绑定，这是下一代平台战争的核心
Anthropic 平台团队两位负责人给出一个被很多人忽略的判断：好的 agent 不是靠通用马具搭模型拼出来的，而是每一个模型都需要和自己深度定制的「马具」紧密配合。模型在进化，Agent SDK、tool calling、context 管理也必须跟着演化。Cursor 那种一层抽象兼容多模型的做法，长远看可能被各家的专有马具拉开差距。

"I think for the next generation of models... everyone's taking slightly different techniques and perspectives on how they want to advance their model. ... The harness and the model get very paired."
— Angela, Head of Platform Engineering, Anthropic
产品范式
SOURCE
AI & I PODCAST
Agent 的「可生产化」比「能干活」难出一个量级
很多人都能用 Claude Code 写一个能跑的原型。但真正让 agent 投入生产、24 小时不停机、处理长时异步任务、管理 session 状态和凭证——这些基础设施问题才是最大的坑。Anthropic 内部也反复踩过，最终决定把 Managed Agents 做成开箱即用的平台层。
▶ 查看原文
"What we saw with a lot of customers was: okay, now I want to go and take that thing and get it into production... and everybody hits an infrastructure wall."
— Caitlin, Head of Platform Product, Anthropic
组织效能
SOURCE
AI & I PODCAST
Agent 需要「主人」—— 没人持续维护的 agent 会死掉
Anthropic 发现最成功的 agent 不是由中央平台团队维护的，而是由业务团队在使用过程中自己迭代——甚至通过 Claude Code 直接改 PR。Agent 的 ownership 应该是去中心化的，平台只需提供「安全的修改通道」。
▶ 查看原文
"A really cool thing ended up happening where people on those teams who were using the tool were like 'I wish this little thing could get tweaked'. And they popped open Claude Code, made some changes themselves to the actual system."
— Caitlin, Head of Platform Product, Anthropic
技术治理
SOURCE
ANTHROPIC BLOG
系统提示词改一行，Claude Code 可能变笨——Anthropic 踩过的坑值得所有人看
Anthropic Engineering 发布罕见详细 postmortem：Claude Code 过去一个月出现三次「变笨」。原因包括：默认推理深度调低、session 过期后 reasoning 被清空、降低冗长 system prompt 影响编码质量。每个改动都是合理优化，叠加起来就像全面降级。

"Because each change affected a different slice of traffic on a different schedule, the aggregate effect looked like broad, inconsistent degradation."
— Anthropic Engineering: "An update on recent Claude Code quality reports"

MATTERS · 议题

Claude 开始「做梦」——空闲时自我检查记忆并修剪矛盾
Peter Yang 总结 Alex Albert 访谈：当 agent 不在执行任务时，会回顾自身记忆、发现矛盾并修剪——这个源自睡眠中记忆处理的「dreaming」过程，可能是 LLM 自主迭代的关键方向。
Claude 新增日常连接器：从 AllTrails 到 Uber Eats
连接器目录突破 200 个。Claude 会在对话中动态推荐 —— 从 work tool 向 everyday assistant 的关键一步。
Sam Altman 罕见夸赞 ChatGPT 最新更新「好太多了」
首次公开承认 GPT-5 有退化问题后，Altman 发推说更新很棒。1.3 万点赞，未透露具体细节。
Everlane 故事：从 Pinterest 竞品到破产再到品牌重生
Linear 产品负责人 Nan Yu 回顾：疫情摧毁了 Everlane 的「通勤装」市场，但 J.Crew、Madewell 都曾死过又重生——品牌的生命力超出想象。
Agent 评估的终极形态：只剩「结果」和「预算」两个参数
Anthropic 平台团队：一切最终压缩为「可验证的结果」和「预算」两个维度，其余全部由平台自动优化。
多 agent 架构开始分化——不同任务需要不同排列组合
Anthropic 观察到 advisor strategy、adversarial generation、分治组合各有专长。关键是把基础设施做成乐高式的。
